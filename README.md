# RAISE/ICARUS Manuscript Analysis Workflow

This repository contains descriptive workflow entrypoints, documentation, validation reports, final approved manuscript figure artifacts, and aggregate outputs for the RAISE/ICARUS health impact assessment manuscript. The controlled harmonized dataset and participant-level data are not included in this repository. Controlled data may be made available upon reasonable request, subject to RAISE/ICARUS data-governance approval, ethics and privacy constraints, and any required data-use agreements.

Aggregate outputs and validation reports support reproducibility of the underlying results. The controlled workflow can reproduce manuscript and supplementary aggregate outputs when run locally with approved access to the controlled harmonized dataset.
