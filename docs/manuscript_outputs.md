# Manuscript Outputs

The manuscript item manifest maps final manuscript and supplementary labels to expected output files, source aggregate files, generation scripts, and validation rules.

Public output names use the final manuscript or supplementary label. Examples include:

- `figures/main/Figure 1 - seasonal_personal_pm_distributions.png`
- `figures/main/Figure 5 - hia_attributable_cases.png`
- `figures/main/Figure 6 - yll_scenario_outputs.png`
- `tables/supplementary/Supplementary Table S3 - Lag-specific heart-rate and stress models.xlsx`
- `results/lag_models/Supplementary Table S3 data - Lag-specific HR stress models.csv`

Each item must pass its validation rule before inclusion in a public aggregate candidate.
