# Manuscript Outputs

The manuscript item manifest maps manuscript and supplementary labels to expected output files, source aggregate files, generation scripts, and validation rules.

The final mapping is generated by `scripts/create_manuscript_item_manifest.py` from the current manuscript and supplementary files. If the current files cannot be identified with confidence, the workflow reports `MANUSCRIPT_ITEM_MAPPING_STATUS = BLOCKED_CURRENT_DOCS_NOT_IDENTIFIED`.

Required output names include the manuscript or supplementary label once detected. Examples include:

- `figures/main/Figure TBD - Seasonal personal PM distributions.png`
- `figures/main/Figure TBD - HIA attributable cases.png`
- `figures/main/Figure TBD - YLL scenario outputs.png`
- `tables/supplementary/Supplementary Table TBD - Analysis-specific denominators.xlsx`
- `results/lag_models/Supplementary Table TBD data - Lag-specific HR stress models.csv`

Each item must pass its validation rule before inclusion in a public aggregate candidate.
