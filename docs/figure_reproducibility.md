# Figure Reproducibility

Final approved figures are included as manuscript output artifacts. Aggregate outputs and validation reports are provided to support reproducibility of the underlying results.

Template fidelity is a validation gate. Each required figure needs a template-fidelity record covering dimensions, aspect ratio, font family, font sizes, legend location, color palette, panel layout, axis-label style, mark style, annotation style, p-value formatting, PM label status, source-device label status, and HIA/YLL significance-marker status.

HIA and YLL figures show scenario estimates and do not use significance markers. Correlation figures include significance annotations where applicable. OLS panels use thresholded small-p notation.

The release candidate preserves the final approved figure files and validates them against the approved figure references. Figure source aggregate data and validation reports are included for auditability.
