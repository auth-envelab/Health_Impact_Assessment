# Data Access

Participant-level data are not public. The controlled harmonized dataset is not included in this repository.

The repository does not include raw data, processed participant-level data, questionnaire records, TAD records, demographic microdata, participant-day rows, participant-night rows, minute-level rows, or model-input rows.

Access to the controlled harmonized dataset is governed by the RAISE/ICARUS controlled-data process. Approved users should run the workflow inside the controlled-data environment and export only safe aggregate outputs, regenerated figures, tables, and validation reports.
